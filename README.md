# Frontend E-commerce App
This is a pure frontend e-commerce app using HTML, CSS, and JavaScript, fully responsive, that works without any server.
All data (users, products, cart, orders) is stored in `localStorage`.

## Features
- User registration and login (simulated via localStorage)
- Product listing
- Cart management
- Checkout with fake payment
- Admin panel: add products, view orders, update order status
- Responsive design
- Works entirely in the browser, can be hosted on GitHub Pages

## How to run
1. Clone the repository or download files.
2. Open `index.html` in any browser.
3. Register or login.
4. Admin: open `admin.html` to manage products and view orders.
